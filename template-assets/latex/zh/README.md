# F.U.C.K Journal LaTeX 模板

这是 **F.U.C.K Journal** 的中文 LaTeX 投稿模板。

F.U.C.K = **Foundations of Universality, Complexity and Knowledge**  
中文可译为：**普遍性、复杂性与知识基础**

## 文件说明

- `FUCKJournalCN.cls`：中文期刊模板类
- `example.tex`：中文示例稿件

## 推荐编译器

- 中文稿件请使用 `XeLaTeX` 或 `LuaLaTeX`

## 前置信息命令

- `\FJtitle{...}`
- `\FJsubtitle{...}`（可选）
- `\FJauthor{...}`
- `\FJaffiliation{...}`
- `\FJabstract{...}`
- `\FJkeywords{...}`
- `\makefjtitle`

## 提示

- 上传到平台的 PDF 应与最终稿内容一致。
- 修订、返修与编辑流转仍建议在平台的结构化投稿空间中完成。
