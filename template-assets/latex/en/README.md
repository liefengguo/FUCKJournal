# F.U.C.K Journal LaTeX Template

This package provides the English LaTeX submission template for **F.U.C.K Journal**.

F.U.C.K = **Foundations of Uncertainty, Complexity, and Knowledge**

## Files

- `FUCKJournal.cls`: Journal class for English manuscripts
- `example.tex`: Example manuscript showing the expected structure

## Recommended compiler

- `pdfLaTeX` for English manuscripts
- `XeLaTeX` also works if your setup prefers it

## Front-matter commands

- `\FJtitle{...}`
- `\FJsubtitle{...}` (optional)
- `\FJauthor{...}`
- `\FJaffiliation{...}`
- `\FJabstract{...}`
- `\FJkeywords{...}`
- `\makefjtitle`

## Notes

- Keep the final manuscript PDF consistent with the version uploaded to the platform.
- Use the online structured submission workspace for revision cycles and editorial handling.
